# Demo data

Sauvegardes `.db` prêtes à restaurer dans Snapfacture (Réglages → Sauvegarde → Restaurer).

## `demo-plomberie-saadi.db`

Données fictives d'un plombier auto-entrepreneur français en franchise TVA.

- **Entreprise** : Plomberie Saadi (Rodrigue Saadi, France, franchise TVA active)
- **Catalogue** : Intervention d'urgence (90 €), Débouchage canalisation (120 €), Remplacement robinet (150 €)
- **7 pièces** étalées de février à mai 2026 :
  - 5 factures B2C (espèces, virement, carte) — clients Sophie Lambert, Mehdi Benali, Camille Roussel, Laura Pereira, Thomas Vasseur
  - 1 facture **B2B** (Syndic Foncia Paris XIII, SIRET client renseigné → mentions B2B automatiques sur le PDF)
  - 1 **avoir** sur la N°1 (motif : prestation annulée)

Toutes les données sont **fictives** : numéros SIRET, IBAN, noms, adresses, téléphones, emails — aucune correspondance avec une vraie entreprise ou personne.

Usage : captures d'écran, vidéos de démo, présentation à un investisseur, tests manuels après refactor.
